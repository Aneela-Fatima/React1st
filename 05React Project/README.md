# 🔐 Password Generator


A simple and interactive password generator built with **React** and **Tailwind CSS**.  
It allows users to generate secure passwords with customizable options in real-time.

- **React (Hooks)**
- **Tailwind CSS**
- **JavaScript (ES6)**

## ⚙️ How It Works

1. User selects password length using the slider  
2. User enables/disables numbers and special characters  
3. Password is generated automatically based on selected options  
4. User can copy the password with a single click
