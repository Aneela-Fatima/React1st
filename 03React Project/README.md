# 🚀 React + Tailwind CSS Learning Project

**Date:** 25 June 202

## 📌 Overview

This project is a simple React application built using Vite where I explored and learned core frontend concepts including:

* Tailwind CSS integration (latest version)
* Component-based architecture in React
* Props usage
* Component reusability

---


## 🎯 What I Learned

### ✅ 1. Tailwind CSS Integration

* Installed Tailwind CSS in a Vite + React project
* Configured Tailwind using the new v4 approach
* Used utility classes directly in JSX
* Understood the difference between Tailwind v3 and v4 setup



### ✅ 2. Components in React

* Broke UI into reusable components
* Understood how React components work
* Created functional components

---

### ✅ 3. Props (Passing Data)

* Learned how to pass data from parent to child components
* Used props to make components dynamic



### ✅ 4. Component Reusability

* Reused the same component with different data
* Made UI more scalable and maintainable

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone <your-repo-link>
```

2. Navigate to project folder:

```bash
cd project-name
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

---

## 🎨 Tailwind Setup (Important)

For Tailwind v4:

```css
@import "tailwindcss";
```

Make sure:

* `index.css` is imported in `main.jsx`
* Vite config includes Tailwind plugin
