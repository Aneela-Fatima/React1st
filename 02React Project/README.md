# React Counter App

**Date:** 24 June 2026  

## 📌 Project Overview
This is a simple React counter application built as a beginner project to understand the basics of React, including components, state management, and event handling.

The app allows users to increase and decrease a counter value with defined limits.

---

## 🚀 Features
- Increment counter value
- Decrement counter value
- Counter range is limited from **0 to 20**
- Prevents value from going below 0 or above 20
- Simple and clean UI

---

## 🧠 Concepts Learned
- React Functional Components
- useState Hook
- Event Handling in React
- Conditional Logic in UI
- State Updates using previous state
