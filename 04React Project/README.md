# 🎨 React Color Changer App

## 📅 Date

June 25, 2026

---

## 📌 Project Overview

This is a simple React project where I built a color changer app. The background color of the screen changes dynamically when different buttons are clicked.

The goal of this project was to practice React fundamentals and understand how state works in real applications.

---

## 🚀 Features

* Change background color with a single click
* Multiple color options available
* Responsive layout using Tailwind CSS
* Clean and simple UI

---


## 🧠 What I Learned

### 🔹 React Concepts

* How to use `useState` hook
* Managing and updating state dynamically
* Handling button click events
* Component structure and rendering

### 🔹 Tailwind CSS

* Using utility classes for styling
* Creating responsive layouts
* Applying flexbox for alignment
* Styling buttons and containers

### 🔹 General Skills

* Writing clean and reusable code
* Debugging errors during setup

---

## ⚙️ How It Works

* A state variable `color` is created using `useState`
* When a button is clicked, `setColor()` updates the state
* The background color of the main div updates instantly
